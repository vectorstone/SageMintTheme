# Changelog

## 0.3.0

- Added the new `Sage Mint Night` dark theme for IntelliJ IDEA 2025.3 Islands UI.
- Added a matching `Sage Mint Night` editor color scheme.
- Kept the original `Sage Mint` theme as the light variant for a complete light/night pair.

## 0.2.0

- Simplified the plugin to a single static Islands Light theme.
- Keeps the fixed mint palette anchored by `#BFE2BE`.
- Keeps a matching static `Sage Mint` editor scheme.

## 0.1.0

- Initial Marketplace-ready release of Sage Mint Theme.
- Adds a light mint UI palette anchored by `#BFE2BE`.
- Adds a matching editor color scheme for readable code surfaces.
